# Aplicación web de indexado de la API de iTunes
## Prerrequisitos:

Asegurarnos de tener Node.js instalado. Podemos descargar Node.js de: https://nodejs.org/

## Instalar dependencias

(Dentro de la carpeta del proyecto):\
```properties
npm install
```  

## Iniciar la aplicación

```properties
npm start
``` 

Abrimos: [http://localhost:3000](http://localhost:3000) para ver la aplicación en nuestro buscador.
